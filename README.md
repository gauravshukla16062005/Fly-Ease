# Fly-Ease: Smart Flight Booking System

## Overview
Fly-Ease is a smart flight ticket booking system that lets users search flights, compare prices, select seats, and book tickets seamlessly. It features real-time availability, secure payments, user accounts, and admin flight management for a smooth travel experience.

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Java 
Database: MySQL
